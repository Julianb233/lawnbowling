# 14-03: Order Fulfillment, Webhooks, Tracking, Club Custom Merch, Affiliate Links

## Scope
- Order fulfillment pipeline: Stripe webhook -> Printify order creation -> production
- Enhanced Stripe webhook to handle one-time shop payments
- Club custom merchandise request form
- Shop navigation update with Custom Merch link
- Affiliate links already built in equipment page (SHOP-08)

## Implementation

### Order Fulfillment (`src/lib/shop/fulfillment.ts`)
- `fulfillOrder()` — Submits orders to Printify after Stripe payment
- `parseOrderItemsFromMetadata()` — Extracts product/variant IDs from Stripe metadata
- Parses customer name, maps shipping address to Printify format
- Sends order to production after creation
- Graceful error handling (order created but production send failed = manual retry via Printify dashboard)

### Stripe Webhook Enhancement (`src/app/api/stripe/webhook/route.ts`)
- Added shop order detection via `metadata.source === "lawnbowl-shop"`
- Extracts shipping details from `collected_information.shipping_details`
- Parses items from metadata and calls `fulfillOrder()`

### Club Custom Merchandise (`src/app/shop/custom-merch/`)
- `page.tsx` — Server page with metadata
- `CustomMerchForm.tsx` — Client form with: club name, contact info, product selection (8 categories), quantity ranges, club colors, notes, logo upload instructions, success confirmation

### Shop Navigation
- Added "Custom" nav link in shop layout header

### Affiliate Links
- Already built in `/shop/equipment` page with links to Henselite, Taylor, Drakes Pride, Aero, Amazon
- No additional work needed (SHOP-08 complete)
