# 14-02: Shop UI (catalog, product detail, cart, Stripe checkout)

## Task
Build the shop UI with product catalog, product detail pages, cart, and Stripe checkout.

## Status: Mostly Pre-existing

The shop UI was largely built in earlier phases. This task focuses on
replacing the checkout placeholder with a real Stripe checkout flow.

## Pre-existing Components (verified)
- `ShopCatalog.tsx` - Product listing with category filter tabs
- `ProductDetail.tsx` - Full product page with color/size selectors
- `ProductCard.tsx` - Product card with image, tags, price
- `CartDrawer.tsx` - Sliding cart drawer with quantity controls
- `AddToCartButton.tsx` - Add-to-cart with success animation
- `cart.ts` - localStorage-backed cart with React hook (useSyncExternalStore)
- `layout.tsx` - Shop layout with sticky header, navigation, cart drawer

## New: Stripe Checkout Flow
- `POST /api/shop/checkout` - Creates Stripe Checkout session:
  - Maps cart items to Stripe line_items with price_data
  - Adds shipping as separate line item
  - Collects shipping address via Stripe (US only)
  - Stores product/variant IDs in metadata
  - Redirects to Stripe-hosted checkout page
- Updated `CheckoutPlaceholder.tsx` with:
  - Order summary with item thumbnails
  - Payment summary with subtotal, shipping, total
  - "Pay with Stripe" button that calls checkout API
  - Free shipping over $75
  - Success page (order confirmed, cart cleared)
  - Cancelled page (friendly message, cart preserved)
  - Empty cart page
  - Loading state with spinner
  - Error handling with user-friendly messages
  - Trust signals (secure payment, shipping timeline)

## Success Criteria
1. Product catalog page with category filters -- DONE (pre-existing)

3. Cart system with add/remove/quantity -- DONE (pre-existing)
4. Stripe checkout for one-time payments -- DONE (new)
