# 09-02: 4-Screen Check-In Flow

## Objective
Implement a clear 4-screen check-in flow with A-Z letter filter, auto-reset, and undo.

## Flow

### Screen 1: Welcome
- Large bowls icon (140px circle)
- "Welcome to Bowls Day" heading
- "Check In Now" primary button (72px, full width)
- Shows count of already-checked-in players
- "Touch anywhere to get started" hint text

### Screen 2: Name Search (A-Z Filter)
- Progress bar showing checked-in count
- "Find Your Name" heading with instruction text
- A-Z letter filter navigation (60px buttons)
  - "All" button to show everyone
  - Letters without players are disabled
  - Active letter highlighted in brand green
  - Handles 300+ member names via surname filtering
- Full-width player rows (72px touch targets)
  - Format: "SURNAME, First"
  - Already checked-in players show green checkmark, disabled
  - Unchecked players show "Check In" button
- Back button to return to Welcome

### Screen 3: Position Select
- "Welcome, {firstName}!" greeting
- 4 position options: Skip, Vice, Lead, Any Position
- Each option: 88px tall, shows position name + description
- Back button returns to Name Search (not Welcome)

### Screen 4: Confirmation
- Large green checkmark (140px circle)
- "You're checked in, {firstName}!" heading
- Position shown
- 10-second undo window with countdown button
- "Change Position" option
- Insurance upsell card (non-blocking)
- 15-second auto-reset countdown bar
- Auto-resets to Welcome screen (not Name Search)
