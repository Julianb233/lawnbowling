# 09-02: 4-Screen Check-In Flow

## Objective
Implement the complete 4-screen kiosk check-in flow for elderly users.

## Screens
1. **Welcome** - Tournament overview, "Check In Now" button, check-in count
2. **Name Search** - A-Z letter filter, full-width player rows, progress bar
3. **Position Select** - Skip/Vice/Lead/Any with descriptions, step indicator
4. **Confirmation** - Checkmark, position display, undo (10s), auto-reset (15s)

## Changes
- Added "welcome" step to CheckInStep type
- Default step is now "welcome" (not "list")
- Reset flow returns to "welcome" screen and clears active letter
- Welcome screen shows bowls icon, greeting, checked-in count, "Check In Now" CTA
- List screen heading changed to "Find Your Name" with instruction text
- Back button added to list screen (returns to welcome)
- Position back button goes to list (not welcome)
- Step indicators show "Step 2 of 3" and "Step 3 of 3"

## Requirements Met
- KIOSK-03: 4-screen check-in flow (Welcome -> Name Search -> Position -> Confirm)
- KIOSK-04: 15-second auto-reset after confirmation
- KIOSK-05: A-Z letter filter for 300 members
- KIOSK-06: 10-second undo window
