# 09-01: Kiosk Component Redesign

## Objective
Redesign kiosk components for 60-80+ year old users with WCAG AAA accessibility compliance.

## Changes

### Touch Targets
- All interactive elements: minimum 56px height (letter filter buttons: 60px)
- Primary action buttons (KioskButton): 72px minimum height
- Position selection buttons: 88px minimum height
- Player list rows: 72px minimum height (up from 72px, maintained)
- Admin PIN keypad buttons: 72px minimum (up from 64px)
- Navigation tabs: 72px minimum (up from 64px)

### Typography
- KioskHeading h1: 40px (up from 36px)
- KioskHeading h2: 34px (up from 28px) -- now meets 32px+ requirement
- KioskHeading h3: 32px (up from 24px) -- now meets 32px+ requirement
- KioskText body: 22px (up from 20px)
- KioskText label: 20px (up from 18px)
- KioskText caption: 18px (up from 16px) -- nothing below 18px
- Position option labels: 28px (up from 26px)
- Position descriptions: 20px (up from 18px)
- Player name in list: 24px (up from 22px)
- Inactivity warning heading: 36px (up from 32px)
- Inactivity warning body: 22px (up from 20px)
- Admin keypad digits: 28px (up from 24px)

### WCAG AAA Contrast (7:1 minimum)
All color combinations validated:
- Primary text (#1A1A1A) on background (#FAFAF5): 16.6:1
- Primary text (#1A1A1A) on white (#FFFFFF): 17.4:1
- Secondary text (#4A4A4A) on background (#FAFAF5): 8.5:1
- Secondary text (#4A4A4A) on white (#FFFFFF): 8.9:1
- Brand green (#1B5E20) on white (#FFFFFF): 7.9:1
- White (#FFFFFF) on brand green (#1B5E20): 7.9:1
- Danger red (#9B1B1B) on white (#FFFFFF): 8.2:1

### Layout
- Header venue name: 40px (up from 36px)
- Confirmation checkmark circle: 140px (up from 120px)
- Countdown bar: h-3 / 240px wide (up from h-2 / 200px)
- Added aria-labels to admin keypad buttons
