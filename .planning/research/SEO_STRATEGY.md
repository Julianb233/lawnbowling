## 1. Keyword Research

### Methodology

Search volume estimates below are derived from industry benchmarks for niche sports terms, cross-referenced against Google Trends relative interest and competitor traffic patterns. Lawn bowling is a niche sport with ~2,800 active members in the US (Bowls USA) and ~58 registered clubs. Search volumes are LOW compared to mainstream sports but HIGH intent -- people searching these terms are genuinely interested in playing or managing the sport.

> **Important context:** "Lawn bowling" is a niche keyword universe. Total monthly US searches across ALL lawn bowling terms likely total 30,000-60,000/month. This is actually an advantage -- low competition means ranking page 1 is achievable within 3-6 months for most terms.

### 3.2 Dynamic Sitemap

Next.js 15 App Router supports dynamic sitemap generation natively.

**File:** `src/app/sitemap.ts`

```typescript
import { MetadataRoute } from 'next'

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  // Static pages
  const staticPages = [
    { url: 'https://lawnbowl.app', changeFrequency: 'weekly', priority: 1.0 },
    { url: 'https://lawnbowl.app/features', changeFrequency: 'monthly', priority: 0.9 },
    { url: 'https://lawnbowl.app/pricing', changeFrequency: 'monthly', priority: 0.9 },
    { url: 'https://lawnbowl.app/clubs', changeFrequency: 'daily', priority: 0.9 },
    { url: 'https://lawnbowl.app/learn/lawn-bowling', changeFrequency: 'monthly', priority: 0.8 },
    { url: 'https://lawnbowl.app/learn/rules', changeFrequency: 'monthly', priority: 0.8 },
    { url: 'https://lawnbowl.app/tournaments/calendar', changeFrequency: 'daily', priority: 0.8 },
    { url: 'https://lawnbowl.app/blog', changeFrequency: 'weekly', priority: 0.7 },
  ]

  // Dynamic: Club pages from database
  const clubs = await getClubs() // Supabase query
  const clubPages = clubs.map((club) => ({
    url: `https://lawnbowl.app/clubs/${club.slug}`,
    lastModified: club.updated_at,
    changeFrequency: 'weekly' as const,
    priority: 0.7,
  }))

  // Dynamic: State pages
  const statePages = getUniqueStates(clubs).map((state) => ({
    url: `https://lawnbowl.app/clubs/${state.slug}`,
    changeFrequency: 'weekly' as const,
    priority: 0.8,
  }))

  // Dynamic: City pages
  const cityPages = getUniqueCities(clubs).map((city) => ({
    url: `https://lawnbowl.app/clubs/${city.slug}`,
    changeFrequency: 'weekly' as const,
    priority: 0.7,
  }))

  // Dynamic: Tournament pages
  const tournaments = await getTournaments()
  const tournamentPages = tournaments.map((t) => ({
    url: `https://lawnbowl.app/tournaments/${t.slug}`,
    lastModified: t.updated_at,
    changeFrequency: 'daily' as const,
    priority: 0.6,
  }))

  // Dynamic: Blog posts
  const posts = await getBlogPosts()
  const blogPages = posts.map((post) => ({
    url: `https://lawnbowl.app/blog/${post.slug}`,
    lastModified: post.published_at,
    changeFrequency: 'monthly' as const,
    priority: 0.6,
  }))

  return [...staticPages, ...statePages, ...cityPages, ...clubPages, ...tournamentPages, ...blogPages]
}
```

## 7. Implementation Priority

### Phase 1: Technical Foundation (Week 1-2)
**Impact: Critical | Effort: Low-Medium**

- [ ] **Create robots.txt**: Block /api/, /admin/, /kiosk/, /auth/

- [ ] **Add default metadata**: Title template, description, OG tags in root layout.tsx
- [ ] **Submit to Google Search Console**: Verify lawnbowl.app, submit sitemap
- [ ] **Submit to Bing Webmaster Tools**: Secondary but free
- [ ] **Set up Google Analytics 4**: Track organic traffic from day 1
