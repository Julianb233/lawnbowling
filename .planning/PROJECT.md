# Pick a Partner

## Description
Progressive Web App for recreational sports clubs where players browse, select, and team up with partners for activities like Pickleball, Lawn Bowling, and Tennis. Runs on iPad (venue kiosk) and iPhone (personal). Features real-time availability board, partner matching, court management, user auth, and integrated liability waiver signing.

## Type
greenfield

## Stack
- Next.js 15 (App Router)
- Tailwind CSS + Radix UI
- Supabase (Postgres + Realtime + Auth + Storage)
- Vercel hosting
- PWA (Serwist/next-pwa)
