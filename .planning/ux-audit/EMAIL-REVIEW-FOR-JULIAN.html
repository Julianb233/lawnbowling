Real-World Research — Key Insight

The app must mirror how clubs have run for decades. 17 sources researched across US, UK, Canadian, and Australian clubs. Flow: Drop name tag → Drawmaster pulls names publicly → Posted on board → Vice scores with pencil → Card to results table → Winners announced at bar.
TERMINOLOGY "Check In" → "Put Your Tag In"
TERMINOLOGY "Generate Draw" → "Do the Draw"
TERMINOLOGY "Score Entry" → "Scorecard"
TERMINOLOGY "Leaderboard" → "Results"
RULE Every action must take 3 taps or fewer
RULE Never use: Dashboard, Settings, Sync, Upload, Configure, Algorithm
