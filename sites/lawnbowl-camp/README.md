# lawnbowl.camp

Standalone microsite for DEI lawn bowling insurance. Co-branded with DEI and Lawnbowling.

## Development

```bash
cd sites/lawnbowl-camp
npm install
npm run dev  # runs on port 3001
```

## Deployment

Deploy to Vercel with the `lawnbowl.camp` domain:

```bash
cd sites/lawnbowl-camp
npx vercel --prod
```

Configure the custom domain in Vercel dashboard: `lawnbowl.camp`

## Features

- DEI insurance landing page with per-session pricing tiers
- Club-wide bulk coverage options
- Quote request form
- FAQ section
- SEO-optimized metadata
- Responsive design (mobile-first)
